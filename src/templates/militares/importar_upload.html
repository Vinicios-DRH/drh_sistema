Importador inteligente de militares

Carregue uma planilha, analise os dados e importe com segurança

Este módulo lê arquivos Excel ou CSV, identifica colunas compatíveis com a tabela de militares, sugere campos complementáveis e permite aplicar OBM fixa antes da gravação definitiva.

Enviar arquivo para análise

O sistema vai validar as colunas, cruzar com os dados existentes e montar uma prévia completa antes de qualquer alteração no banco.

Formatos aceitos: .xlsx, .xls e .csv.
Como funciona a análise
1. A planilha é lida e as colunas são normalizadas.
2. O sistema tenta localizar militares por CPF, matrícula, RG ou nome.
3. Você escolhe os campos que deseja importar.
4. A importação pode complementar dados vazios ou sobrescrever dados existentes.
